🏗️ Architektur

7 Komponenten
SIELAFF SUe2020 ──MDB-Bus──▶ QIBIXX PI HAT PLUS ──UART──▶ RASPBERRY PI 5
📡
MDB Sniffer
Liest MDB-Bus ueber Qibixx Pi Hat Plus (UART /dev/ttyAMA0, 115200 Baud). Dekodiert alle MDB-Befehle passiv.
🧠
CEO Agent
Hybrid-Agent: Regelengine + LLM (Mistral). Prueft alle 5 Min auf Anomalien, erstellt Tages-/Wochenberichte.
📊
Sales Analytics
Berechnet Peak Hours, Produktgeschwindigkeit und Preisempfehlungen aus SQLite-Daten.
🌤️
Weather Manager
Holt alle 2h Wetterdaten von OpenWeatherMap und korreliert mit Verkaeufen.
📱
Telegram Bot
Sendet Alerts + Berichte. Empfaengt Befehle: /status /report /stock /sales /peak /weather
🔗
n8n Webhook
Sendet Events an n8n fuer weitere Automatisierung (E-Mail, Slack, etc.).
🖥️
Flask Dashboard
Web-UI auf Port 5000. REST-API + Live-Dashboard. Zugriff ueber Tailscale.
DATENFLUSS: MDB-Bus → Parser → SQLite → Analytics/Agent → Dashboard/Telegram/n8n
